Try to split string to list of numbers

Use ',' as separator, extra spaces will be deleted, each gotten token will be checked as number or as empty value at all

Examples:

  • "" - "" (Ok)
  • "123,456" - "123|456" (Ok)
  • "123 456" - "123456" (Ok)
  • " 123, 456 " - "123|456" (Ok)
  • "abc,123" - (Fail)
